# Personal Portfolio

A responsive personal portfolio built using React, Vite, and Styled Components. The website showcases my projects, skills, and contact information with support for light and dark themes.

## Live Demo

https://venkarcharan.github.io/venkat-portfolio/

## Tech Stack

- React
- Vite
- Styled Components
- React Router DOM
- GitHub Pages
- GitHub Actions

## Installation

```bash
git clone https://github.com/venkarcharan/venkat-portfolio.git
cd venkat-portfolio
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project is deployed using GitHub Pages and GitHub Actions.
